Sumuqan

https://github.com/SvenFrankson/sumuqan

Sumuqan is a BabylonJS-based library providing methods to procedurally animate insect-like creatures.

It's what powers the Phasmida you see walking in the background.

Sumuqan est une extension pour BabylonJS qui offre une méthode d'animation procédurale pour des créatures insectoides ou robotique.

Les phasmes visibles en arrière-plan sont animés avec Sumuqan.

Mummu

https://github.com/SvenFrankson/mummu

Mummu is a collection of tools for BabylonJS applications. Typescript-written, it targets a wide range of domains, from BABYLON.VertexData manipulation to Vector3, Quaternion and Spatial Intersections.

The intersection and collision code that is used for Physic in MarbleRunSimulator is in project Mummu.

Mummu est une collection d'outils pour des applications utilisant BabylonJS. Ecrite en Typescript, elle couvre un large champ d'application, de la manipulation de BABYLON.VertexData à des méthodes de géométrie (Vector3, Quaternion et Intersections Spatiales)

Le système de collision et d'intersection utilisé dans Marble Run Simulator est dans le projet Mummu.

Nabu

https://github.com/SvenFrankson/nabu

Nabu is a Typescript written collection of simple tools for number manipulation (easing methods, simple maths) and data structures.

Nabu est une collection d'outils simple pour la manipulation de nombre (easing) et de structures de données.
